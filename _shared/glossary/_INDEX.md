# _shared/glossary/_INDEX.md

**공통 용어집** — 여러 SI 프로젝트에서 공통으로 사용하는 용어 정의.

## 📚 용어집 카테고리

| 카테고리 | 파일 | 설명 |
|---|---|---|
| SI 일반 | `si-common.md` | SI 방법론, 산출물 용어 |
| 환경 구분 | `environments.md` | DEV/STG/PRD, 검증/운영 표기 |
| 아키텍처 | `architecture.md` | MSA, API, 인프라 공통 용어 |

> 위 파일들은 예시. 실제 필요 시 생성.

## 🔤 기본 표기 원칙

- **환경 구분**: `개발환경(DEV) → 검증환경(STG) → 운영환경(PRD)`
- **산출물 버전**: `v1`, `v1.1`, `v2` 형식 (semver 아님, 간단 증가)
- **영문 약어**: 최초 등장 시 풀이 병기
- **날짜**: `YYYY-MM-DD` 고정

## 🎯 고객사별 용어

각 프로젝트별 고유 용어는 해당 프로젝트의 `docs/references/glossary.md` 에.
**여기(_shared)에는 올리지 말 것** — 혼동 유발.
