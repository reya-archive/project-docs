# _shared

여러 SI 프로젝트에서 **재사용**할 공통 자산.

## 📂 구조

```
_shared/
├── glossary/        # 공통 용어집 (SI 일반, MCNC 내부 용어)
│   └── _INDEX.md
└── patterns/        # 재사용 가능한 템플릿, CLAUDE.md 조각, 체크리스트
    └── _INDEX.md
```

## 🎯 활용 방식

각 프로젝트의 `CLAUDE.md` 에서 이 폴더의 자산을 참조:

```markdown
## 🎯 작업 시 참고할 공통 자산

<shared_assets>
- `../../_shared/glossary/` — 공통 용어집
- `../../_shared/patterns/` — 재사용 템플릿
</shared_assets>
```

## 📝 이 폴더에 넣을 것 / 넣지 말 것

### ✅ 넣을 것
- 여러 프로젝트에서 반복되는 SI 표준 용어
- MCNC 내부 표기 규칙 (환경 구분, 문서 버전 체계 등)
- 재사용 가능한 ADR 템플릿, 회의록 템플릿
- CLAUDE.md 의 공통 규칙 조각 (인용해서 쓸 수 있는)

### ❌ 넣지 말 것
- 특정 고객사 전용 용어 (각 프로젝트 내부에)
- 프로젝트 고유 의사결정
- 민감한 내부 정책 (별도 사내 위키로)
