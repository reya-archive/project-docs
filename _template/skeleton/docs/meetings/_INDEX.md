# docs/meetings/_INDEX.md

**Tier 2 — 회의록 타임라인**

시간순으로 누적. 가장 최근 회의가 상단.

## 🗓️ 회의록 목록

> 실제 회의록 추가 시 이 표 갱신.

| 날짜 | 회의체 | 주제 | 파일 |
|---|---|---|---|
| YYYY-MM-DD | {회의명} | {핵심 주제} | `{파일명}` |
| YYYY-MM-DD | {회의명} | {핵심 주제} | `{파일명}` |

## 📝 회의록 작성 규칙

**파일명**: `YYYYMMDD_회의체명_주제.md`
예: `20260415_주간회의_요구사항검토.md`

**필수 섹션**:
1. 일시 / 장소 / 참석자
2. 안건
3. 논의 내용 (발언자별 또는 주제별)
4. **결정 사항** — 중요한 결정은 `../decisions/` 에 별도 ADR로 승격
5. 액션 아이템 (담당자 / 기한)

## 🔍 Claude 탐색 힌트

사용자가 회의 관련 질의 시:
- "지난주" → 최근 7일 내 파일
- "O월" → 해당 월 파일 필터
- 특정 주제 → 제목(주제 부분) 기반 검색

결정 사항만 필요하면 `../decisions/` 를 먼저 확인하는 것이 효율적.
