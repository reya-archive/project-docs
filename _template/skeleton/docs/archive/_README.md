## 🚫 Claude 동작 규칙

<claude_rule>
이 폴더는 사용자가 **명시적으로 요청할 때만** 읽어야 함.

다음 키워드가 질의에 포함될 때만 탐색:
- "이전 버전", "구버전", "예전", "폐기", "archive"
- 특정 과거 날짜 명시 (예: "2026년 1월 요구사항")

일반 질의 (예: "요구사항이 뭐야?")에 대해서는 이 폴더를 절대 읽지 말 것.
현재 유효 문서는 `../specs/` 에 있음.
</claude_rule>

## 📦 이관 규칙

새 버전이 `../specs/` 에 추가되면:
1. 구버전 파일을 여기로 이동
2. 파일명 prefix에 `DEPRECATED_` 또는 이동 날짜 추가 권장
3. `../specs/_INDEX.md` 버전 이력에 이관 사실 기록

## 🗑️ 삭제 기준

- 프로젝트 종료 후 1년 경과한 archive 파일은 정리 검토
- 법적/감사 목적으로 보존 필요한 것만 남기고 나머지는 삭제
