## 📜 버전 이력

구버전은 `../archive/` 로 이동. 여기에는 현재 유효 버전만 유지.

- YYYY-MM-DD: 요구사항정의서 v1 → v2 (변경점: {요약})
- YYYY-MM-DD: 아키텍처 설계서 v1 (신규)

## 📝 작성 규칙

- 파일명: `YYYYMMDD_문서유형_v버전.md`
- 큰 변경 시 버전 증가, 작은 수정은 파일 내 변경이력만
- 폐기 시 `../archive/` 로 이동 후 이 인덱스에서 제거
